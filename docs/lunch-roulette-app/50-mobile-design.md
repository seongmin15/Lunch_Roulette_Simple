## 3. 화면 상세

### 홈 화면

> 앱 진입점으로, 현재 위치 기반 주변 식당 목록과 룰렛 시작 버튼을 표시한다.

**주요 인터랙션:**

- 앱 실행 시 자동으로 현재 위치를 감지하고 주변 식당을 불러온다.
- 필터 버튼을 눌러 필터 설정 화면으로 이동한다.
- 룰렛 시작 버튼을 눌러 룰렛 화면으로 이동한다.

**연동 API:**

- 카카오/네이버 지도 API - 주변 식당 조회

**UI 상태:**

- 로딩 중
- 식당 목록 표시
- 위치 권한 거부됨
- 검색 결과 없음

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| RestaurantListCard | 개별 식당의 이름, 거리, 카테고리를 카드 형태로 표시 |
| FilterChip | 현재 적용된 필터 조건을 태그 형태로 표시 |
| RouletteStartButton | 룰렛 실행을 시작하는 주요 액션 버튼 |

---

### 필터 설정 화면

> 가격대, 거리, 음식 카테고리 등 식당 필터 조건을 설정한다.

**주요 인터랙션:**

- 슬라이더로 최대 거리(반경)를 설정한다.
- 가격대 옵션을 선택한다.
- 적용 버튼으로 필터를 저장하고 홈으로 돌아간다.


**UI 상태:**

- 기본 필터 상태
- 필터 수정 중

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| DistanceSlider | 검색 반경을 미터 단위로 조절하는 슬라이더 |
| PriceRangeSelector | 가격대(저렴, 보통, 비싼)를 선택하는 버튼 그룹 |

---

### 룰렛 화면

> 필터링된 식당 목록을 대상으로 룰렛을 돌려 무작위로 식당을 선택한다.

**주요 인터랙션:**

- 룰렛 애니메이션이 실행된다.
- 결과 식당이 강조 표시된다.
- 다시 돌리기 버튼으로 재실행한다.
- 결과 확인 버튼으로 식당 상세 화면으로 이동한다.


**UI 상태:**

- 룰렛 회전 중
- 결과 표시

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| RouletteWheel | 식당 목록을 원형 룰렛으로 시각화하고 회전 애니메이션을 제공 |
| ResultCard | 선택된 식당 이름을 강조하여 표시 |

---

### 식당 상세 화면

> 룰렛으로 선택된 식당의 상세 정보(이름, 주소, 거리, 영업시간)를 표시한다.

**주요 인터랙션:**

- 지도 앱으로 길 안내를 시작한다.
- 홈으로 돌아간다.

**연동 API:**

- 카카오/네이버 지도 API - 식당 상세 정보 조회

**UI 상태:**

- 정보 표시
- 영업시간 정보 없음

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| RestaurantDetailCard | 식당 이름, 주소, 전화번호, 영업시간을 정리하여 표시 |
| NavigationButton | 외부 지도 앱으로 길 안내를 연결하는 버튼 |

---

### 히스토리 화면

> 최근 룰렛으로 선택된 식당 10건의 기록을 시간 순으로 표시한다.

**주요 인터랙션:**

- 히스토리 항목을 탭하여 식당 상세 화면으로 이동한다.
- 전체 히스토리를 삭제한다.


**UI 상태:**

- 히스토리 있음
- 히스토리 없음

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| HistoryListItem | 날짜, 식당 이름을 한 줄로 표시하는 리스트 아이템 |
