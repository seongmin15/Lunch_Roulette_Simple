## 3. 화면 상세

### 홈 화면

> 앱 진입점으로, 현재 위치 기반 주변 식당 목록과 룰렛 시작 버튼을 표시한다.

**주요 인터랙션:**

- 앱 실행 시 자동으로 현재 위치를 감지하고 주변 식당을 불러온다.
- 필터 버튼을 눌러 필터 설정 화면으로 이동한다.
- 룰렛 시작 버튼을 눌러 룰렛 화면으로 이동한다.

**연동 API:**

- 카카오 로컬 Category Search API - 선택된 타입(식당 FD6 / 카페 CE7)으로 1km 반경을 2×2 rect 그리드로 분할하여 각 셀 자동 페이지네이션 수집 (최대 180곳, API 4~12회). Haversine 공식으로 클라이언트 사이드 거리 계산.

**UI 상태:**

- 로딩 중
- 식당 목록 표시
- 위치 권한 거부됨
- 검색 결과 없음

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| SegmentedButton | 식당/카페 PlaceType 전환 토글 |
| RestaurantListCard | 개별 식당의 이름, 거리, 카테고리를 카드 형태로 표시 |
| FilterChip | 현재 적용된 필터 조건을 태그 형태로 표시 |
| RouletteStartButton | 룰렛 실행을 시작하는 주요 액션 버튼 |

---

### 필터 설정 화면

> 거리와 음식 카테고리 필터 조건을 설정한다.

**주요 인터랙션:**

- 슬라이더로 최대 거리(반경)를 100m 단위로 설정한다 (100m~1km, 클라이언트 사이드 필터링).
- 음식 카테고리를 멀티셀렉트로 선택한다 (한식/중식/일식/양식/분식/치킨/피자). 카페 모드에서는 카테고리 섹션이 숨겨진다.
- 적용 버튼으로 필터를 저장하고 홈으로 돌아간다.
- 초기화 버튼으로 기본값(1km, 전체 카테고리)으로 복원한다.


**UI 상태:**

- 기본 필터 상태
- 필터 수정 중

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| DistanceSlider | 검색 반경을 100m 단위로 조절하는 슬라이더 (100m~1km, 클라이언트 필터링) |
| CategorySelector | 음식 카테고리(7종)를 FilterChip 멀티셀렉트로 선택 (식당 모드에서만 표시) |

---

### 룰렛 화면

> 필터링된 식당 목록을 대상으로 슬롯머신을 돌려 무작위로 식당을 선택한다.

**주요 인터랙션:**

- 슬롯머신 스핀 애니메이션이 실행된다 (easeOutCubic 감속).
- 결과 식당이 센터 하이라이트 바에 강조 표시된다.
- 다시 돌리기 버튼으로 재실행한다.
- 결과 확인 버튼으로 식당 상세 화면으로 이동한다.
- 공유 버튼으로 결과를 외부에 공유한다.


**UI 상태:**

- 슬롯머신 회전 중
- 결과 표시

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| SlotMachine | ListWheelScrollView 기반 슬롯머신. 식당 이름/카테고리/거리를 표시하고 루핑 스핀 애니메이션 제공 |
| ResultCard | 선택된 식당 이름을 강조하여 표시 |

---

### 식당 상세 화면

> 룰렛으로 선택된 식당의 상세 정보(이름, 주소, 거리, 영업시간)를 표시한다.

**주요 인터랙션:**

- 지도 앱으로 길 안내를 시작한다.
- 홈으로 돌아간다.

**연동 API:**

- 카카오 로컬 API - 식당 상세 정보 (검색 결과에 포함된 데이터 활용)

**UI 상태:**

- 정보 표시
- 영업시간 정보 없음

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| RestaurantDetailCard | 식당 이름, 주소, 전화번호, 영업시간을 정리하여 표시 |
| NavigationButton | 외부 지도 앱으로 길 안내를 연결하는 버튼 |

---

### 히스토리 화면

> 최근 룰렛으로 선택된 식당 10건의 기록을 시간 순으로 표시한다.

**주요 인터랙션:**

- 히스토리 항목을 탭하여 식당 상세 화면으로 이동한다.
- 전체 히스토리를 삭제한다.


**UI 상태:**

- 히스토리 있음
- 히스토리 없음

**컴포넌트:**

| 컴포넌트 | 역할 |
|---------|------|
| HistoryListItem | 날짜, 식당 이름을 한 줄로 표시하는 리스트 아이템 |
